# PUG
  * templating engine used for server-side templating
  * example: 
  ```jade
    div
      h1 My heading
      ul
        li Some list item
        li Some other list item
      p This here is a paragraph.
  ```
  * result:
      ```html
      <div>
        <h1>My heading</h1>
        <ul>
          <li>Some list item</li>
          <li>Some other list item</li>
        </ul>
        <p>This here is a paragraph.</p>
      </div>
      ```

## Use in Express.js
